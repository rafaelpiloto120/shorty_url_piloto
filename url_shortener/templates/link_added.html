About Shorty URL

This is a simple project developed with the goal to learn and practice the use of Python and Flask in building web apps. Just for fun I'll try to keep it up and running as long as possible. However, I can't guarantee stability in high intensity usage, once this this is hosted at a free Heroku dino.

Flask | Python

This project is based on Flask, a microframework for Python used for building web apps. I recommend this video for the ones interested in developing a similar app.

Bootstrap v4.3.1

As front-end I used Bootstrap framework, one of the most popular open-source front-end frameworks for responsive web development.
